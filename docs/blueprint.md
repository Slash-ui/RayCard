# **App Name**: Luminous Border

## Core Features:

- Glassmorphism Box: Create a glassmorphism effect for a container with a frosted glass appearance.
- Dynamic Light Source: Implement JavaScript to track the mouse cursor position and simulate a light source.
- Border Highlighting: Adjust the box-shadow and border glow of the container based on the mouse position, creating a shining effect.

## Style Guidelines:

- Primary color: Light blue (#B0E2FF) to enhance the glassy effect.
- Background color: Very light gray (#F0F8FF) with low opacity to blend with the background.
- Accent color: Soft lavender (#E6E6FA) to complement the light blue.
- Font: 'Inter', a sans-serif font suitable for clean, modern UIs.
- Use a centered layout for the glassmorphism box to draw focus.
- Smooth transition for the box-shadow and border glow to provide a fluid response to mouse movement.
